# Sistema de Gestión Documental en Python

Aplicación desarrollada en Python que permite gestionar documentos mediante un sistema CRUD básico con persistencia de datos en archivos JSON.

## Funcionalidades

- Agregar documentos
- Listar documentos
- Buscar documentos por nombre
- Persistencia de datos en archivo JSON

## Tecnologías utilizadas

- Python
- Manejo de archivos JSON
